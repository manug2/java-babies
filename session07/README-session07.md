
Numeric data type conversions

<br/>
1. Type following commands using `jshell` to understand number data types 

`byte b = Byte.MIN_VALUE;`

`short s = Short.MAX_VALUE;`

`char c = 'A';`

`int i = 123456789;`

`double d = 1.23456789;`

`float f = 1.23456789f;`

`long lo = i * i;`

`double d = Double.MIN_VALUE;`

Down cast explicit

`byte b = (byte) i;`

Up cast automatic

`int i = b;`

char to int and back

`int i = 'a'`

`char c = (char) i;`
